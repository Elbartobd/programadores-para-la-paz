Se envió una petición POST a http://localhost:3000/registro con el siguiente JSON:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor respondió con:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor recibió correctamente los datos enviados en el cuerpo de la petición
y los devolvió organizados en formato JSON, confirmando que req.body funciona
correctamente con express.json().
